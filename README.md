# CodeSense: Rule-Based Algorithm Analyzer (Python)

A static analysis engine that identifies algorithm patterns and estimates time complexity using Python AST.

## Requirements
- Python 3.10+
- No external dependencies

## How to Run

```bash
py -3.10 -m venv venv
venv\Scripts\activate
python sample.py
```
